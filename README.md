# Mahwa Desktop Application


Mahwa Desktop Application is a JavaFX-based login and registration system developed for managing user authentication and credentials. The application leverages JavaFX for building the user interface and Scene Builder for UI design. SQLite is utilized as the database to store user data securely.

## Features

- **Login System:** Secure authentication system allowing users to log in with their credentials.
- **Registration:** User-friendly registration process for new users to create an account.
- **JavaFX UI:** Modern and intuitive user interface designed using JavaFX technology.
- **SQLite Database:** Efficient SQLite database integration for storing user data and credentials securely.

## Usage

1. Clone the repository to your local machine.
2. Open the project in your preferred Java IDE.
3. Build and run the application.
4. Use the login page to authenticate or register as a new user.

## Technologies Used

- JavaFX
- Scene Builder
- SQLite
